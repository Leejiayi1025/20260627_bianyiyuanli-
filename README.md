# 编译原理 · 知己

编译原理课程复习系统 —— 知识点阅读 + 章节答题 + 错题本

## 功能

- **复习**：17 章知识点完整覆盖，卡片式阅读，默认展开
- **答题**：每章 6-10 道题，作答后即时出解析，支持收藏题目
- **错题本**：自动收录错题，支持错题重练
- **导出 PDF**：一键导出全部知识点为 PDF

## 在线使用

## 本地使用

双击打开 `编译原理·知己.html` 或使用本地服务器：

```bash
npx serve .
```

## 技术栈

纯 HTML + CSS + JavaScript 单页应用，无依赖，无框架。
